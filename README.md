# Bookmarker App

A simple web application to save and manage your favorite websites.

The app allows you to add, view, update, delete, and search bookmarks.  
All data is stored in Local Storage, so bookmarks remain after page refresh.

## Features

- Add website name and URL
- Visit saved websites
- Update existing bookmarks
- Delete bookmarks with confirmation
- Search bookmarks by name
- Form validation
- Local Storage support

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Font Awesome
- SweetAlert2

## How to Run

Open the `index.html` file in your browser.

## Validation Rules

- Site name must be at least 3 characters
- Site URL must be a valid URL

Frontend CRUD Application
